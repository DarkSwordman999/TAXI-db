# База данных TAXI

## О проекте
Реляционная база данных для учёта работы службы такси. Содержит информацию об автомобилях, водителях, клиентах, поездках и тарифах.

## Технологии
- **PostgreSQL 16** - основная СУБД
- **Bash** - скрипты для автоматизации
- **Git** - контроль версий

## Структура базы данных

### Таблицы

| Таблица | Описание | Количество записей |
|---------|----------|-------------------|
| АВТОМОБИЛИ | Информация об автомобилях (марка, модель, класс, госномер) | 30 |
| ВОДИТЕЛИ | Данные водителей (ФИО, телефон, стаж, рейтинг) | 32 |
| КЛИЕНТЫ | Данные клиентов (ФИО, телефон) | 250 |
| ПОЕЗДКИ | Информация о поездках (дата, стоимость, связи) | 1060 |
| ТАРИФЫ | Цены за км и минуту по классам | 4 |
| СЕЗОН | Сезоны для аналитики | 12 |

### Классы автомобилей
- эконом
- комфорт
- бизнес

## Команды для работы

```bash
./h task1      # выручка по сезонам
./h task2      # выручка по сезонам и классам авто
./h task3      # выручка по дням недели и водителям
./h task4 N    # выручка по водителям за день N

./h 200        # создать функции
./h 201 порог  # водители с выручкой выше порога
./h 202        # выручка по классам и сезонам
./h 203        # сводная таблица (водители × классы)
./h 204        # выполнить всё
```
### Файлы проекта

- **h** - основной скрипт для запуска запросов
- **schema/tables/** - создание таблиц
- **data/** - вставка данных
- **queries/** - SQL-запросы по задачам
- **queries/lab7/** - функции
